# Regex FSM Lab

Реалізація скінченного автомата для обробки регулярних виразів на Python з використанням патерна **State**.

## Функціонал
* **Символи:** Підтримка англійської абетки (`A-Z`, `a-z`), цифр та крапки (`.`), що означає будь-який символ.
* **Оператори:** `*` — нуль або більше повторень (квантифікатор зірочка).
    * `+` — одне або більше повторень (квантифікатор плюс).
* **Архітектура:** Кожен елемент виразу є окремим об'єктом-станом.

## Як це працює
* Програма розбиває регулярний вираз на окремі класи (`AsciiState`, `DotState`, `StarState` тощо).
* Реалізовано пріоритетну перевірку наступних станів. Автомат спочатку намагається "вийти" з циклу (`*` або `+`), а якщо це неможливо — зациклюється. Це дозволяє коректно обробляти складні вирази типу `.+`.
* Перевірка завершується успішно лише тоді, коли автомат проходить через усі символи стрічки та опиняється у фінальному стані `TerminationState`.
